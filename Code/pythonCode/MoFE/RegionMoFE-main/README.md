<h2 id="bYAPB">Quick Start</h2>
CITY: NY, Chi

TASK: crime, checkIn, serviceCall, pop

1. Interpretable Universal Representation Learning.

```bash
python RegionMoFE_train.py --city CITY
```

2. Prompt4RE

```bash
cd Prompt4RE
python PromptLearning.py --city CITY --task TASK
```
